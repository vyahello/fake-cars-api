Fake vehicles 🚍 🚘 REST API

A RESTful API service for fake vehicles followed by API Star 🌟 framework!

API endpoints:

  • Get list of all vehicles:
    GET /rest
    curl -X GET localhost:5000/rest
  • Create a single vehicle:
    POST /rest
    curl -X POST -H "Content-type: application/json -d '{"manufacturer": "BMW", "model": "X5", "year": 2011, "vin": "WBA"}'
  • Get single vehicle:
    GET /rest/101
    curl -X GET localhost:5000/rest
  • Update single vehicle:
    PUT /rest/101
    curl -X PUT -H "Content-type: application/json -d '{"manufacturer": "Toyota", "model": "Camry", "year": 2014, "vin": "ECA"}'
  • Delete single vehicle:
    DELETE /rest/101
    curl -X DELETE localhost:5000/rest/99
